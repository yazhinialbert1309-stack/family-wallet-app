# 👨‍👩‍👧‍👦 Family Wallet - Smart Budget Tracker

A secure, real-time family expense tracking application built with **Next.js 14** and **Supabase**. This project was designed to help families manage their daily spends with a clean dashboard and member-specific tracking.

## 🚀 Features
- **Real-time Updates:** Instantly syncs expenses across all family devices.
- **Member Breakdown:** Track spending for individual members (Appa, Amma, Brother, Me).
- **Interactive Dashboard:** Visual cards for Total Spends and Member Totals.
- **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop.
- **Smart History:** Categorized expense list with icons and delete functionality.

## 🛡️ Security Implementation (Cybersecurity Focus)
As a Cybersecurity trainee, I have implemented the following security measures:
- **Row-Level Security (RLS):** Configured Supabase policies to control data access.
- **Environment Variables:** Securely managed API keys using `.env` to prevent leakage.
- **Input Validation:** Sanitized user inputs for amounts and categories.

## 🛠️ Tech Stack
- **Frontend:** Next.js (React), Tailwind CSS, Lucide React (Icons).
- **Backend/DB:** Supabase (PostgreSQL).
- **Hosting:** Vercel.
